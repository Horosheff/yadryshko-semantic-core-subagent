# Примеры запуска Core

## Минимальный запуск

```text
/core https://example.ru
```

## Хороший запуск для российского сайта

```text
/core https://example.ru/
Регион: Россия, Wordstat ID 225.
Цель: заявки с сайта.
Приоритет: Яндекс, Google вторично.
Исключить: вакансии, скачать бесплатно, нерелевантные бренды.
```

## Локальный бизнес

```text
/core https://example.ru/
Регион: Москва, Wordstat ID 213.
Цель: заявки на услугу.
Приоритет: Яндекс.
Конкуренты:
- https://competitor-1.ru/
- https://competitor-2.ru/
```

## Интернет-магазин

```text
/core https://shop.example.ru/
Регион: Россия 225.
Цель: продажи категорий и карточек товара.
Приоритет: Яндекс + Google.
Нужно: кластеры категорий, фильтры, URL map, P0/P1 контент-план.
```

## Если Wordstat ещё не подключён

```text
/core проверь, видишь ли ты MCP-KV Wordstat tools. Если Wordstat недоступен, не выдумывай частотности и напиши, что нужно подключить по docs/mcp-kv-wordstat-setup.md.
```
